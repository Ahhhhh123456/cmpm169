Experiment 5 - 3D Graphics

Description

This is my 3D Graphic experiment. Press Mouse1 to destroy the generated 4 towers. The height of all the towers should be randomly generated.

Technical

It uses Canon.js to have physics for the boxes.

Reflection

In this project, I wanted to simulate and earthquake using physics. But that ended up changing when I thought it would be way cooler to have the towers just break when the mouse button is clicked. I had no clue how to implement canon.js so I'm very glad ChatGPT was able to help me acheieve what I wanted.
